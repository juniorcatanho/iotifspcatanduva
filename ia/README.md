# Inteligência Artificial

Script responsável pela análise inteligente dos dados do orquidário.

## Abordagem
- Sistema especialista baseado em regras
- Isolation Forest (não supervisionado)
- Camada de explicabilidade (XAI)

## Entrada
- MQTT: estufa/dados/telemetria

## Saída
- MQTT: estufa/ia/recomendacoes

## Observação
As configurações de acesso devem ser definidas pelo usuário.
